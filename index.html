Twitter data gathering using R

In this assignment, I will be using R studio with various libraries such as twitteR, ROAuth to access the Twitter API. First I will run a "handshake" to authenticate the Twitter API with R. Furthermore, I will use the function "searchTwitter" from the library(twitteR) to fetch tweets on a specific topic. This would give me a string list which I can apply "twListToDF" to convert to a data frame table. From that, I can use internal functions from R such as hist() or plot() to render a graph. Once all the plots have been set, they will be converted into a PDF document with all the plots and source code for verification
